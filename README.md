### Installing Medora

To install **Medora**, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/rolandaayo/medora-landing-page.git
```

Windows:

```bash
git clone https://github.com/rolandaayo/medora-landing-page.git
```
